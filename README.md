# VNA sprite demo (2D)

Apri `index.html` in un browser.

## Contenuti
- `assets/truck_base.png` : sprite top-down del carrello (base).
- `assets/fork.png` : sprite forche (separato) ruotabile DX/SX.

## Note
- I parametri (Corridoio/Posto/Livello/Scaffale) sono quelli che avevi indicato.
- Il livello non cambia la 2D (top-down), ma viene mantenuto nel comando/target.
- Il carrello segue un percorso Manhattan (prima allinea su Y, poi su X).

## Sostituzione sprite
Puoi rimpiazzare le PNG in `assets/` mantenendo i nomi file.
